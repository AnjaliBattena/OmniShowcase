## 🎯 How to Use

### Starting the Application
1. The app opens to the **Home Page** showing all 9 templates
2. Each template is displayed as a card with:
   - Preview image
   - Category badge
   - Description
   - Technology tags
   - Hover effects

### Navigation
- **Click any template card** to view the full screen
- **Use the "Back" button** on each page to return home
- **Direct URL access**: Type any path (e.g., `/voyago`) to go directly to that screen
